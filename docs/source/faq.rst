PiPower Proが動作しないのはなぜですか？
---------------------------

初めてバッテリーを入れるときや、バッテリーを外して再度取り付けるとき、
バッテリーが正常に動作しない場合があります。


これは、バッテリーが取り外されたとき、
オンボードの過放電保護回路のメカニズムにより、
電圧が低すぎると判断され、保護回路が作動するためです。


この時、 **Type C** ケーブルを充電ポートに差し込み、保護回路をリリースする必要があります。
そうすれば、バッテリーは正常に使用できます。
